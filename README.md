# Remote Score

A React Native application built with Expo for remote score management.

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Expo CLI
- For Android: Android Studio and Android SDK
- For iOS: Xcode (macOS only)

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd remote-score
```

2. Install dependencies:
```bash
npm install
```

## Running the App

### Start the development server:
```bash
npm start
```

### Run on Android:
```bash
npm run android
```

### Run on iOS:
```bash
npm run ios
```

### Run on Web:
```bash
npm run web
```

## Technologies Used

- React Native 0.81.5
- Expo SDK ~54.0
- React 19.1.0
- Expo Status Bar
- React Native KeyEvent
